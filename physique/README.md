## Base de math pour physique

### Fonction affine

Une fonction f\(x\) est affine si elle est de forme $$f(x)= ax+b$$ avec :  
- a le **coefficient directeur** ou la **pente**  
- b est **l'ordonnée à l'origine**  
Elle est représentée par une droite
